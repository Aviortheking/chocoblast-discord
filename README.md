# TCGdex BOT

Browse the Pokemon Trading Card game cards using the TCGdex BOT

## Commands

| Command  | Description                            |
|----------|----------------------------------------|
| find     | Find a card by it's name name          |
| findbyid | Find a card by it's global/local ID id |
| help     | Show the commands registered           |
